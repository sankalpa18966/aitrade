================================================================
QUICK REFERENCE — WHAT EACH FILE DOES
================================================================

config.py
  Stores all settings and API keys.
  Change settings here - other files read from here.

data_collector.py
  Fetches raw data from internet.
  ForexFactory + Reuters + RSS feeds.
  Output: raw news + events dictionary.

analyzer.py
  Takes raw data, sends to Gemini AI.
  Gemini analyzes sentiment per currency.
  Output: currency scores + drivers + mood.

signal_generator.py
  Takes currency scores.
  Ranks strongest vs weakest.
  Builds formatted Telegram message.
  Output: ready-to-send signal message.

telegram_bot.py
  Simple: takes message, sends to your phone.
  Nothing else.

scheduler.py
  Runs everything automatically at right times.
  6AM brief, 15min pre-event, 6PM summary.
  Runs 24/7 on VPS.

main.py
  Master file. Calls all others in sequence.
  Run this to test full pipeline manually.
